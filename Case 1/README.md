A. PERSYARATAN SISTEM (PREREQUISITES) -
-------------------------------------
1. Pastikan "Go" (Golang) sudah terinstal di komputer Anda.
   Cek dengan perintah terminal: go version

B. CARA MENJALANKAN PROGRAM (RUN)
---------------------------------
Ada dua cara untuk menjalankan program ini:

CARA 1: Input Manual (Copy-Paste)
1. Buka Terminal / Command Prompt di folder ini.
2. Jalankan perintah:
   go run BE_Pemrog_Khalifaalhasan.go
3. Kursor akan berkedip menunggu input.
4. Copy data dari bagian "C. DATA INPUT" di bawah, lalu Paste ke terminal.
5. Tekan ENTER.
6. (Penting) Tekan Ctrl+Z (Windows) atau Ctrl+D (Mac/Linux) lalu ENTER untuk melihat hasil.

CARA 2: Input Menggunakan File (Disarankan)
1. Buat file bernama 'input.txt' berisi data input.
2. Jalankan perintah:
   go run BE_Pemrog_Khalifaalhasan.go < input.txt

C. DATA INPUT (SAMPLE SOAL)
---------------------------
Silakan gunakan data berikut untuk pengujian:
5
100
103
109
1002
2049
